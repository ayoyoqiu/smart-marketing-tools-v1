super_admin (超级管理员) → 拥有所有三种角色权限
    ├─ super_admin 权限 ✅
    ├─ admin 权限 ✅
    └─ user 权限 ✅

admin (管理员) → 拥有两种角色权限
    ├─ admin 权限 ✅
    └─ user 权限 ✅

user (普通用户) → 只有基础权限
    └─ user 权限 ✅

👤 普通用户 (张三)
   ✓ 创建自己的Webhook地址
   ✓ 发送推送任务
   ✓ 查看自己的消息历史
   ✓ 上传图片生成链接
   ✗ 看不到其他用户数据
   ✗ 无管理功能

👨‍💼 管理员 (李四)
   ✓ 所有普通用户功能
   ✓ 查看所有用户数据
   ✓ 管理用户账号
   ✓ 帮助用户排查问题
   ✓ 可切换到普通用户视角

👑 超级管理员 (王五)
   ✓ 所有管理员功能
   ✓ 最高系统权限
   ✓ 系统配置和维护
   ✓ 可在三种角色间切换

🔍 如何查看自己的角色？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法1: 左侧菜单 → 个人资料 → 查看"当前角色"
方法2: 右上角显示当前用户和角色
方法3: 能看到"管理员功能"菜单 = 您是管理员
